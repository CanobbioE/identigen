IDENTIGEN

Identigen è un programma che genera dati di un'identità fittizia completamente casuali.

Tali dati non sono correlati tra loro e passano tutti i controlli di forma.

I dati di seguito generati sono interamente fittizi.

API

È possibile utilizzare direttamente le API esposte all'indirizzo identigen.herokuapp.com.
I parametri che possono essere utilizzati sono i seguenti:

dateformat
Il formato delle date. Supporta: 'eu','us','ja' (default "eu")
fields
I campi da stampare, devono essere separati da una virgola. Usare 'all' per stamparli tutti (default "all")
format
Il formato dell'output desiderato. Supports: 'json', 'csv', 'human' (default "human")
maxage
L'età massima dell'identità da generare. Deve essere positiva e maggiore di minage. (default 55)
minage
L'età minima dell'identità da generare. Deve essere positiva e minore di maxage. (default 25)
n
Il numero di identità da generare. Deve essere positivo e minore di 15. (default 1)

Per esempio il seguente link restituisce una lista di due identità con nome, cognome in formato 'human':
identigen.herokuapp.com/?format=human&fields=nome,cognome,datadinascita&n=2
