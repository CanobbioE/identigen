IDENTIGEN

Identigen è un programma che genera dati di un'identità fittizia completamente casuali.

Tali dati non sono correlati tra loro e passano tutti i controlli di forma.

I dati di seguito generati sono interamente fittizi.

API

È possibile utilizzare direttamente le API esposte all'indirizzo identigen.herokuapp.com.
I parametri che possono essere utilizzati sono i seguenti:

dt_fmt
Il formato delle date. Supporta: 'eu','us','ja' (default "eu")
format
Il formato dell'output desiderato. Supporta: 'json', 'csv', 'xml', 'human' (default "human")
maxage
L'età massima dell'identità da generare. Deve essere positiva, minore di 200 e maggiore di minage. (default 55)
minage
L'età minima dell'identità da generare. Deve essere positiva, minore di 200 e minore di maxage. (default 25)
number
Il numero di identità da generare. Deve essere positivo e minore di 15. Per ottenere più di 15 identità per volta si prega di installare il software. (default 1)
fields
I campi da stampare, devono essere separati da una virgola. Usare 'all' per stamparli tutti (default "all"). I possibili valori sono: Nome Cognome Gender PaeseDiNascita ProvinciaDiNascita Indirizzo NumeroDiTelefono DataDiNascita CodiceFiscale PartitaIva ComunePartitaIva Documento Patente CartaDiCredito Iban Username Password

Per esempio il seguente link restituisce una lista di tre identità con nome, cognome in formato 'human':
https://identigen.herokuapp.com/?format=human&fields=Nome,Cognome&number=3
